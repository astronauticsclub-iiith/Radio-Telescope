#### Targets at (16-24MHz, using Dipole Antenna):  
- The Sun: Radio bursts and flares
- Jupiter: Powerful radio storms and bursts from its magnetic field interacting with its moon Io
#### Targets at (1420MHz, using Parabolic Antenna):
- Neutral hydrogen in the Milky Way (https://youtu.be/Y1N-_HOpQNo)
- Bright radio sources
